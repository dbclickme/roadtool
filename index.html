🛣️ 道路摊铺助手 - 使用说明
📁 文件要求

Excel文件必须包含以下列：

  • 里程：桩号位置，支持 K5+520, DK11+200 等格式
  • 偏距：测量点距离中线的距离（米）
  • 目标高：实测高程（米）
  • 边桩高(层底高)：设计高程（米）
示例数据：
里程: DK11+740, 偏距: 0, 目标高: 17.623, 边桩高: 17.124
里程: DK11+740, 偏距: 0, 目标高: 16.964, 边桩高: 17.124
同一偏距的两个点会自动配对：高程高的为钢钎顶(挖)，低的为地面(填)
📊 挂线计算

差值 = 目标高 - 边桩高 (正值=挖，负值=填)

挂线位置(cm) = 向下量 - 向上量 × (松铺系数 - 1) - 挂线高

步骤：上传文件 → 设置参数 → 点击"计算" → 查看结果 → 导出Excel

🧱 填料计算

压实方计算（已扣除虚铺）

  • 填料厚度 = 边桩高 - 目标高 (正值=填方)
  • 断面宽度 = 最大偏距 - 最小偏距（或手动输入）
  • 断面面积 = Σ(填料厚度 × 分段宽度) —— 梯形法
  • 本段填料量 = (前一断面面积 + 本断面面积) / 2 × 桩号间距
⚠️ 注意事项
  • 同一偏距点需要两个高程值才能计算挂线位置
  • 填料计算只计算填方数据，按压实方计
  • 本程序使用全球统一CDN，保证稳定访问
